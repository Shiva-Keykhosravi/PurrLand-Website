# PurrLand Website

A cozy and cute website dedicated to everything your cat needs!  
This project showcases products, care tips, and happy stories for cat lovers.

## Features

- Responsive design for desktop and mobile
- Interactive hamburger menu on mobile
- Product showcase section (Portfolio)
- Customer comments/testimonials
- Newsletter subscription section
- Clean, cozy and modern CSS styling

## Tech Stack

- HTML5
- CSS3 (Flexbox, Responsive)
- JavaScript (for mobile menu toggle)
- Font Awesome for icons
